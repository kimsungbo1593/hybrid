# 김성보 포트폴리오

개인 웹 포트폴리오 사이트입니다.  
HTML과 CSS를 활용하여 자기소개, 학력/경력, 그리고 외부 링크(깃허브 등)를 담은 간단한 홈페이지를 제작했습니다.  

---

## 📌 주요 기능
- **홈페이지 소개**
  - 상단 네비게이션 바 (소개, 인적, 학술, 경력 탭)
  - 자기소개 및 프로필 이미지
- **기술 스택**
  - HTML5, CSS3
  - 웹 표준 구조 설계
  - 반응형(기본적인 레이아웃 대응)
- **참조 링크**
  - GitHub, 개인 링크 등 연결
