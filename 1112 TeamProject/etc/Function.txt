홈
- 빠른 기록 -> 버튼 3개로 바로 navbar로 들어갈 필요 없이 바로 기능 사용 가능
- 오늘의 추억 -> 성장 앨범에 있는 사진 중 랜덤하게 사진 하나 표시


성장 앨범
- 카메라 버튼 -> 카메라 권한 획득 후 사진 혹은 동영상 촬영 후 업로드 가능
- 상세 보기 -> 사진의 정보 확인 및 삭제 가능

성장 데이터
- 데이터 입력 -> 측정일 / 키 / 몸무게 선택 및 작성 후 저장
- 성장 그래프 -> 입력된 데이터를 조회 및 삭제 가능, 입력된 데이터를 기반으로 성장 곡선 표시

일정
- 추가 버튼 -> 일정 제목 / 날짜 및 시간 / 알림 설정 후 저장
- 일정 상세 -> 일정 정보 확인 및 삭제 가능

설정
-> 아기 이름 / 생일 / 성별 설정 후 저장 시 홈에 저장된 아기 이름 표시
